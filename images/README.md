## 使用说明

1. 所有图标建议使用PNG格式，背景透明
2. 分享图片建议使用JPG或PNG格式
3. 表情图片可以使用GIF格式以实现动画效果
4. 图片文件大小应控制在合理范围内，建议单个图标不超过10KB
5. 分享图片建议不超过100KB

## 注意事项

- 确保图片文件名与代码中引用的名称一致
- 图片路径应相对于项目根目录
- 建议使用专业的图片处理工具优化图片大小和质量
